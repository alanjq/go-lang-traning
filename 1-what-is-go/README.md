# What is Go

Is a statically typed high-level language.

Designed by Google

It has garbage collection (like JAVA), memory safety and structural typing.

- Statically typed
- Class based
- Object oriented

## Creation aspects
- Simplicity
- Safety
- Readability

## Support offered by Go
- Procedural
- Concurrent
- Distributed

# Core aspects of Go

- Similar to JAVA
- Modern language
- General purpose
- Open Source
- Influenced by C
